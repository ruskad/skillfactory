# Выбор топпинга v2

**Цель:** выбрать топпинг при заказе пиццы.

**Акторы:** авторизованный пользователь, фронтенд, бэкенд.

**Контекст:** при заказе пиццы на сайте пользователь должен иметь возможность выбрать топпинг для определенных видов пиццы.

**Сценарий использования:**

1. Авторизованный пользователь выбирает пиццу для заказа.
2. Фронтенд отправляет запрос в бэкенд для проверки, можно ли для данной пиццы выбрать топпинг и варианты.
3. Фронтенд отправляет ответ по возможности выбора топпинга с вариантами.
4. Бэкенд отображает пользователю,  можно ли выбрать топпинг и какой.
5. Пользователь выбирает топпинг.
6. Фронтенд отправляет запрос на бэкенд для определения стоимости пиццы с учетом топпинга.
7. Бэкенд пересчитывает стоимость пиццы и отправляет стоимость на фронтенд.
8. Фронтенд отображает итоговую стоимость.
9. Пользователь оформляет заказ.

**Альтернативный сценарий:**

1. Пользователь выбрал топпинг для пиццы, но не оформил заказ.
2. Пользователь открывает заказ и изменяет топпинг для пиццы.
3. Фронтенд отправляет запрос на бэкенд для пересчета заказа.
4. Бэкенд пересчитывает стоимость пиццы и отравляет стоимость на фронтенд.
5. Фронтенд отображает стоимость пиццы.
6. Пользователь оформляет заказ.
